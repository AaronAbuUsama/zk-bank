Color Selection Guide

Semantic Colors (consistent across themes):

  • Price Up / Profit:Green variant
  • Price Down / Loss / Liquidation:Red variant
  • Warning / Attention:Yellow/Orange variant
  • Info / Accent:Cyan/Blue variant
  • Primary Action:Theme's primary color

Flush Grid Implementation:

  • Background colors separate sections (banner, main, footer)
  • No box borders - color contrast creates structure
  • Liquidations use left border + background tint for emphasis
  • Position preview uses dashed border (only exception)
